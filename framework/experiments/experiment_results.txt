================================================================================
METRIC DEFINITIONS
================================================================================
  Compilable%    - Compilation Success Rate: % of forms that produce
                   compilable WinUI 3 output (estimated)
  Complete%      - Migration Completeness: % of source controls
                   that were successfully migrated to WinUI 3
  UI Parity%     - UI Parity Score: % of UI properties (colors, fonts,
                   layout) correctly mapped to WinUI 3 equivalents
  Speedup        - Time Reduction Ratio: how many times faster the
                   automated migration is vs estimated manual effort
  Err/100LOC     - Error Density: number of remaining errors per
                   100 lines of generated code (lower is better)
